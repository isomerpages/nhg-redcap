<h4><strong>WHAT THIS IS ABOUT</strong></h4>
<ul data-tight="true" class="tight">
<li>
<p>The NHG-REDCap Production Server will be shutting down between <strong>17th to 21st Apr 2026</strong> to
facilitate server upgrading and maintenance works.</p>
</li>
<li>
<p>This announcement is to inform all NHG-REDCap Users of the pending Shutdown
period, how this will impact their Production Server Projects and Data
Collection activities and what Users need to do to prepare for the Shutdown
Period.</p>
</li>
<li>
<p>We urge all REDCap Users, in particular REDCap Project Owners (ie: Study
PIs), take note of the Shutdown period and plan their research data collection
activities around it.</p>
</li>
</ul>
<h4><strong>WHEN THIS WILL HAPPEN</strong></h4>
<ul data-tight="true" class="tight">
<li>
<p>The Production Server Shutdown Period will begin <strong>from <u>6pm on Friday 17th Apr 2026</u></strong> and
will end at <strong><u>10am, Tuesday 21st Apr 2026</u></strong>, after which
the Production Server will resume normal operations.</p>
</li>
</ul>
<div class="isomer-image-wrapper">
<img style="width: 60%;" height="auto" width="100%" alt="" src="/images/Content Images/RC_Shutdown_Apr_2026.png">
</div>
<h4><strong>HOW USERS WILL BE AFFECTED</strong></h4>
<ul data-tight="true" class="tight">
<li>
<p>There will be <strong><u>NO USER ACCESS</u></strong> to the <strong><u>PRODUCTION SERVER</u></strong> during
the Shutdown period.</p>
</li>
<li>
<p>All Production Server activities will <strong><u>NOT BE AVAILABLE</u></strong> to
Users,<em> </em>including<em>:</em>
</p>
<ul data-tight="true" class="tight">
<li>
<p>Data Entry</p>
</li>
<li>
<p>Project Design and Editing</p>
</li>
<li>
<p>Project Change Requests and Implementation</p>
</li>
<li>
<p>New Account Requests, and New Account Activation Forms<em> (for both Production and Staging Servers)</em>
</p>
</li>
<li>
<p>Suspended Account Re-Activation <em>(for both Production and Staging Servers)</em>
</p>
</li>
<li>
<p>New Project Requests</p>
</li>
</ul>
</li>
</ul>
<h4><strong>WHAT USERS NEED TO DO</strong></h4>
<ul data-tight="true" class="tight">
<li>
<p>Users will need to <strong>assess</strong>, <strong>plan</strong> and <strong>implement</strong> alternative
manual data capture workflows if there is a need to continue with data
collection activities during the shutdown period.</p>
</li>
<li>
<p>Please download and read the <strong>Shutdown Guide</strong> for details
on what Users <strong>WILL NEED TO KNOW, WHAT TO PREPARE FOR</strong>, and
what the available mitigation measures are for the Shutdown Period.</p>
</li>
<li>

Intranet/HVPN access required)</p>
</li>
</ul>
